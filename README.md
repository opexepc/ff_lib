# ff_lib
ff simple file-config library  
check example for using

## step 0 - check software
you must have CMake
## step 1 - build lib
```bash
git clone https://github.com/opexepc/ff_lib.git
cd ff_lib
mkdir build && cd build
cmake ..
make
```

## step 2 - build example
```bash
cd ..
cd example
mkdir build && cd build
cmake ..
make
```
